Immobiliser and transponder
The immobiliser allows the vehicle to be started only if the rider uses the special ignition switch which sends the identification code to a decoder in the instrument panel. The transmission-receiving system consists of a TRANSPONDER in the ignition keys and an antenna in the vicinity of the ignition switch.
Keys with transponder
The two keys contain the secret code used for programming the immobiliser and allowing the engine to be started.
The transponder is fitted inside the key body and consists of a power supply circuit, a ROM (Read-Only Memory) and a transmission circuit. The transponder is supplied with power by the magnetic field generated by the antenna which is enabled as soon as the instrument panel is switched on. Since the transponder is very compact and needs no external battery power supply, it is moulded permanently into the key body.
Warning
The circuits in the key bodies are not shockproof and may therefore be damaged by violent impact.
Immobiliser antenna
The antenna consists of a plastic-coated copper coil connected to a cable that terminates with a connector.
The antenna (A) is fitted onto the ignition switch cover (B), which is specifically modified for the purpose to create a seat in which the antenna is inserted and restrained.
Immobiliser operating principle
Upon key-on (switching on the instrument panel by turning the key) the decoder (incorporated in the instrument panel) activates the antenna, which emits an electromagnetic field that serves to power the transponder in the key body. The transponder responds by transmitting the code that is stored in its internal ROM memory, and which is picked up by the antenna and relayed to the decoder. Here the transponder code is compared to a second code. If the outcome of the comparison is positive, engine starting is enabled.
Warning light signals and immobiliser indications
Whenever the ignition key is turned from ON to OFF, the immobiliser system activates the engine start protection system.
When the engine is started and the key is turned from OFF to ON, the system generates the following signals:
-
if the code is recognised, the protection system releases the engine lock. When you press the START button (A), the engine starts;
-
if the code is not recognised, the “Vehicle/Engine Diagnosis" (10) light comes on; when you press button (2) “t” the page with the message “Error IMMO” and the error code indicating the type of IMMO error will be displayed. In this case, turn the ignition key back to OFF and then to ON again. If the engine still does not start, try again with the other black key.
Removing the immobiliser
Set the ignition switch to OFF.
Disconnect the antenna wiring connector (A) from the main wiring.
Loosen the screws (1), and remove the tank cover (2).
Loosen the screws (3) on the ignition switch cover (4), and remove it.
Slide out the immobiliser antenna (5) from ignition switch.
Refitting the immobiliser
Refitting is the reverse of removal; in particular, tighten the screws (3) and (1) to the specified torque (Sect. C 3, Frame tightening torque values).
